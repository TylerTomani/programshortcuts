Mac Essential Shorcuts

1. Open / Switch / Close Applications

1. Spotlight Search

⌘ + ␣ (space)
DON't Click to open apps
Press Command (⌘) + Space (␣)

Type name of app, press enter when name appears

2. Switch Applications

⌘ + ⇥ (Tab)
Do NOT us the mouse to change programs
Press Command (⌘) + Tab (⇥)

Type name of app, press enter when name appears

3. Quit Applications

⌘ + Q
When a application is open, the name appears in top left
Press Command (⌘) + Q

To quit the current opened application
